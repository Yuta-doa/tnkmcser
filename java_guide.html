Minecraft JAVA版 - サーバーの入り方

JAVA版サーバーに接続する方法

  1. **Minecraft JAVA版(1.21.4)を起動**
    Minecraftのランチャーを開き、最新バージョンまたはサーバー対応バージョンを選択してゲームを起動します。
  2. **「マルチプレイ」を選択**
    メインメニューから「マルチプレイ」をクリックしてください。
  3. **新しいサーバーを追加**
    「サーバーを追加」ボタンをクリックし、以下の情報を入力します：
    • サーバー名: お好きな名前を入力
    • サーバーアドレス: 147.185.221.25:10288
  4. **サーバーに接続**
    追加したサーバーを選択し、「サーバーに参加」をクリックしてゲームを開始します。

接続時の注意事項

もしサーバーに接続できない場合は、以下を確認してください：

  • インターネット接続が安定していることを確認。
  • サーバーアドレスが正しいか再確認。
  • サーバーの推奨バージョンを使用しているか確認。

さらにサポートが必要な場合は、こちらのサポートページをご覧ください。
